# @b2bsim/web

The SvelteKit web app of the B2B Simulator monorepo — SSR UI, API routes, Drizzle/SQLite,
auth, and the email seam. It consumes the `@b2bsim/engine` and `@b2bsim/content` packages.

See the root [README.md](../../README.md) for an overview and setup instructions, and
[ARCHITECTURE.md](../../ARCHITECTURE.md) for the design write-up.
